# Retail Sales Analytics Dashboard  

This project showcases a complete end-to-end Retail Sales analysis using **Power BI**, including data cleaning, KPI creation, data visualization, and insight generation.  
The dashboard provides a clear overview of business performance across Sales, Profit, Quantity, Regions, Products, and Time.

---

## Dataset Description  
The dataset contains retail transactional data with the following fields:

- Order ID  
- Date  
- Month Name  
- Month Number  
- Region  
- Product  
- Category  
- Sales  
- Profit  
- Quantity  

The dataset was cleaned and processed using **Power Query**.

---

## Data Cleaning & Transformation (Power Query)

The following steps were performed:

- Removed duplicate rows  
- Standardized column names  
- Converted data types (Date, Whole Number, Decimal)  
- Created **Month Name** and **Month Number**  
- Sorted Month Name correctly using *Sort by Column*  
- Filled missing data where needed  
- Ensured numerical fields were correctly formatted  
- Applied data validation checks  

These steps ensured the data was ready for accurate analysis and reporting.

---

## Key Performance Indicators (KPIs)

| KPI | Description |
|-----|-------------|
| **Total Sales** | Total revenue generated |
| **Total Profit** | Total profit after cost deductions |
| **Quantity Sold** | Total units sold |
| **Profit Margin %** | Profitability metric = Profit / Sales |

These KPIs provide a quick snapshot of overall business performance.

---

## Visuals Included in the Dashboard

The Power BI report contains the following visuals:

- **KPI Cards** (Sales, Profit, Quantity, Margin)  
- **Sales Trend Over Time** (Line Chart)  
- **Sales by Region** (Bar/Column Chart)  
- **Sales by Category**  
- **Top 10 Products by Sales**  
- **Filters/Slicers**  
  - Region  
  - Category  
  - Month  
  - Product  

The dashboard is clean, interactive, and designed with a modern layout.
